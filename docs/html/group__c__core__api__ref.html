ASTRA_API astra_status_t astra_terminate ( )

Terminate the library, destroying all its streams.

ASTRA_API astra_status_t astra_update ( )

Drive the library to get a new frame of stream.

call this function in while loop until there is a new frame, using sleep to block cpu for several ms in while loop is recommended.
