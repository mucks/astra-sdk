System Requirements

OS Requirements Details
Windows
  • Windows 8 and 10, 32-bit and 64-bit
  • x86-based processor @ 1.8+ ghz
  • USB 2.0
  • 4 gigabytes of RAM
  • Supports development with both Visual Studio 2013 and Visual Studio 2015.
Linux
  • Ubuntu 14.04 or later
  • x86-based processor @ 1.8+ ghz
  • USB 2.0
  • 1 gigabyte of RAM
  • GCC 4.9 and Clang 3.5 have both been extensively tested, but any compiler with support for C++11 should be compatible,including XCode 5 or better, GCC 4.9 or better, Clang 3.1 or better, and Android NDK r9e or better.
Android
  • Android OS 4.4.2 (KitKat) +
  • Multi-core ARMv7a-based processor @ 1.5 ghz+
  • USB 2.0 host support (OTG-capable)
  • 512 megabytes of RAM
  • GCC 4.9 and Clang 3.5 have both been extensively tested, but any compiler with support for C++11 should be compatible,including XCode 5 or better, GCC 4.9 or better, Clang 3.1 or better, and Android NDK r9e or better.
macOS
  • OS X 10.8 or later
  • x86-based processor @ 1.8+ghz
  • USB 2.0
  • 4 gigabytes of RAM
  • GCC 4.9 and Clang 3.5 have both been extensively tested, but any compiler with support for C++11 should be compatible,including XCode 5 or better, GCC 4.9 or better, Clang 3.1 or better, and Android NDK r9e or better.
  • Astra SDK v2.0 for macOS is coming soon. For now you can download Astra SDK v0.5.0 for macOS.

Setup for Visual Studio

When you develop with Visual Studio on a Windows platform, you have some extra work to do.The Astra SDK works fine depends on the OpenNI SDK and Orbbec sensor driver for Windows.So, you should install them rightly first.

  1. Download OpenNI SDK and install it.
  2. Download Orbbec Sensor Driver for Windows and install it.
  3. Download Astra SDK according to your Visual Studio distribution.Unzip it to a folder, for example called $ASTRA_HOME.
  4. Open a new project or an existing one.
  5. In the Visual Studio menu, open the Project menu and choose Project properties.
  6. In the C/C++ section, under the General node, find the "Additional Include Directories" and add "$ASTRA_HOME\include".
  7. In the Linker section, under the General node, find the "Additional Library Directories" and add "$ASTRA_HOME\lib","$ASTRA_HOME\lib\Plugins".
  8. In the Linker section, under the Input node, find the "Additional Dependencies" and add astra.lib,astra_core.lib,astra_core_api.lib

Setup for Android Studio

  1. Download Astra SDK package for Android.
  2. Open a new project or an existing one.
  3. Copy the downloaded file to libs folder
  4. Add the following code to build.gradle
    android {
    repositories {
    flatDir {
    dirs 'libs'
    }
    }
    }
    dependencies {
    implementation (name:'astra-android-release-v2.0.12-b48cd2945b', ext:'aar')
    }
    Note
    The value 'astra-android-release-v2.0.12-b48cd2945b' above is exactly the name of the downloaded file.
