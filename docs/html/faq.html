How to apply for license

C and C++ - Add this directly after calling astra_initialize() and replace <LICENSE STRING> with the provided license key:

orbbec_body_tracking_set_license("<LICENSE STRING>");

Java - Add this directly after calling Astra.initialize(), add import com.orbbec.astra.BodyTracking at the top of the .java file, and replace <LICENSE STRING> with the provided license key:

BodyTracking.setLicense("<LICENSE STRING>");

C# and Unity - Add this directly after calling Astra.Context.Initialize() and replace <LICENSE STRING> with the provided license key:

Astra.BodyTracking.SetLicense("<LICENSE STRING>");
